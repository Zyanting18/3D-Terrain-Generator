# 3D-Terrain-Generator
Users interactive 3D Terrain.
Type make to run.


	           README             
	+----------------------------+
	|  w  : Toogle Wireframe Mode|
	|-----+----------------------|
	|  l  : Toggle Light Mode    |
	|-----+----------------------|
	|  s  : Toggle Flat Shadding |
	|-----+----------------------|
	|  r  : Random using Circle  |
	|-----+----------------------|
	|  R  : Random using Fault   |
	|-----+----------------------|
	|  t  : Draw in Triangle     |
	|----------------------------|
	|  y  : Draw in Quads        |
	|----------------------------|
	|  0  : Switch to Light 0    |
	|----------------------------|
	|  1  : Switch to Light 1    |
	|----------------------------|
	| uio : Control Light Up     |
	|-----+----------------------|
	| hjk : Control Light Down   |
 	|-----+----------------------|
	|  q  : Quit Program         |
	+----------------------------+
